# 🚀 DS Meta - 

Aplicação fullstack para visualização e gerenciamento de dados de
vendas, com backend em Spring Boot e frontend em React.

## 🖥️ Preview

Frontend: (adicione seu link do Netlify)\
Backend API: (adicione seu link do Render)

## 📌 Funcionalidades

-   Listagem de vendas\
-   Filtro por período\
-   Integração com API REST\
-   Deploy em nuvem

## 🛠️ Tecnologias

### Backend

-   Java 17
-   Spring Boot
-   PostgreSQL

### Frontend

-   React
-   Axios

### DevOps

-   Render
-   Netlify

## ⚙️ Como rodar

### Backend

cd backend\
./mvnw spring-boot:run

### Frontend

cd frontend\
npm install\
npm start

## 🔗 Configuração

REACT_APP_API_URL=http://localhost:8080

## ☁️ Deploy

### Render (backend)

-   Criar Web Service
-   Configurar banco PostgreSQL

### Netlify (frontend)

-   Build: npm run build
-   Publish: build ou dist
